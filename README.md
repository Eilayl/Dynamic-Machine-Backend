This is the FastAPI (Python) backend of the system.
It provides the API and schema definitions that the frontend uses to dynamically generate forms and tables.

How to Run
git clone https://www.github.com/eilayl/Dynamic-Machine-Backend
cd Dynamic-Machine-Backend
uvicorn main:app --reload

What I Learned

How to use SQL Models to clearly define object structures in the database.

Gained deeper experience with FastAPI, including models, schemas, and endpoints.

Understood how a small change in the backend models automatically propagates to the frontend without breaking the system.
